# Bank REST API

Проект для управления банковскими картами и пользователями.

## Технологии
- Java 17+
- Spring Boot
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Liquibase
- Docker / Docker Compose
- Swagger (OpenAPI)
